# Operadores Aritiméticos

O operador '+' é o operador de adição numérica. 
As expressões abaixo darão resultados numéricos, pois o php tentará converte-los para numério.

2 *+* 2

```php
$num1 = 2 + 2;     #= 4; int
$num2 = "2" + "2"; #= 4; int

var_dump($num1);
var_dump($num2);
```

## Operadores

- '+' -> Adição
- '-' -> Subtração
- '*' -> Multiplicação
- '/' -> Divisão real
- '%' -> Modulo
- '**' -> Potenciação
